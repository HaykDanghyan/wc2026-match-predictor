# 🏆 World Cup 2026 Match Predictor

Machine learning model that predicts FIFA World Cup 2026 match scores, win/draw/loss probabilities, and group stage qualification using XGBoost Poisson regression and Monte Carlo simulation.

🔗 **[Live App](https://wc2026-match-predictor-777.streamlit.app/)**

---

## Overview

This project builds a full football prediction pipeline from scratch — from raw international match data to a deployed web app — covering data cleaning, feature engineering, model training, and an interactive Streamlit UI.

The core idea: instead of classifying match outcomes directly (which struggles with draws), we train two XGBoost models to predict each team's expected goals (λ), then derive win/draw/loss probabilities and scoreline distributions analytically using the Poisson distribution. Group stage qualification probabilities are computed via Monte Carlo simulation across all 12 World Cup groups.

---

## Features

- **Single match prediction** — expected goals, most likely scoreline, win/draw/loss probabilities, and full scoreline probability matrix for any of the 48 WC teams
- **Group stage simulation** — 5,000 Monte Carlo simulations across all 12 groups, accounting for points, goal difference, and the "best 8 third-placed teams" rule
- **Full qualification ranking** — all 48 teams ranked by probability of advancing to the Round of 32
- **Interactive UI** — built with Streamlit, dark football-themed design

## How It Works

### Data
- **Source:** [International Football Results 1872–2024](https://www.kaggle.com/datasets/martj42/international-football-results-from-1872-to-2017) (Kaggle)
- Filtered to matches from 2002 onward (~23,000 matches)
- Team names standardized using `former_names.csv` + manual mapping

### Feature Engineering
| Feature | Description |
|---|---|
| `home_elo` / `away_elo` | Rolling Elo rating computed from all match results (K=20, home advantage=60) |
| `elo_diff` | Elo rating difference between the two teams |
| `home_form_gf` / `away_form_gf` | Average goals scored over each team's last 5 matches |
| `home_form_ga` / `away_form_ga` | Average goals conceded over each team's last 5 matches |
| `neutral` | Whether the match is played on neutral ground |
| `match_importance` | Tournament type grouped into 4 levels (0=Friendly, 1=Regional, 2=Qualifier, 3=Major) |

### Model
- **Two XGBoost regressors** with `objective='count:poisson'`
  - `home_goal_model` → predicts λ_home (expected goals for team A)
  - `away_goal_model` → predicts λ_away (expected goals for team B)
- From λ_home and λ_away, the full scoreline probability matrix is computed using the Poisson PMF
- Win/draw/loss probabilities are derived by summing the relevant cells of the matrix

### Why Poisson regression instead of classification?
All four classification models tested (Logistic Regression, Random Forest, Gradient Boosting, XGBoost) achieved ~57–60% accuracy but had near-zero draw recall (2–12%). Draws don't occupy a distinct region in feature space, so classifiers default to ignoring them. The Poisson approach naturally produces realistic draw probabilities whenever both teams have similar predicted goal rates.

### Monte Carlo Simulation
Each group has 4 teams playing 6 matches. Whether a team qualifies depends not just on their own results but on all other group matches and tiebreakers (goal difference, goals scored). For the 2026 format, the best 8 third-placed teams across all 12 groups also advance — making closed-form qualification probabilities impractical. Monte Carlo simulates all 12 groups 5,000 times by sampling real integer scorelines from each match's Poisson distribution.

## Model Performance

### Classification baselines (Win/Draw/Loss)
| Model | Accuracy | Macro F1 | Draw Recall |
|---|---|---|---|
| Logistic Regression | 0.60 | 0.45 | 0.02 |
| Random Forest | 0.59 | 0.48 | 0.09 |
| Gradient Boosting | 0.60 | 0.45 | 0.02 |
| XGBoost | 0.57 | 0.47 | 0.12 |

### Poisson goal regression (XGBoost)
| Target | MAE | Poisson Deviance | Avg Predicted vs Actual |
|---|---|---|---|
| Home goals | 1.057 | 1.247 | 1.669 vs 1.635 |
| Away goals | 0.862 | 1.178 | 1.120 vs 1.147 |

> Note: MAE ~1 goal is typical for football prediction — most of this error is the inherent randomness of the sport. Published benchmarks for football outcome prediction land at 53–58% accuracy; our ~57–60% is in range for a model trained purely on match results without player-level data.

## Running Locally

```bash
# clone the repo
git clone https://github.com/YOUR_USERNAME/wc2026-match-predictor.git
cd wc2026-match-predictor

# install dependencies
pip install -r requirements.txt

# run the app
streamlit run wc2026_app.py
```

To retrain the models, open `World_Cup.ipynb` in Jupyter and run all cells. The final cell exports `wc2026_models.pkl` which the app loads at startup.

## Limitations

- Predictions are based purely on historical match results — no player-level data (injuries, suspensions, squad depth), no betting market signals, no weather or travel factors.
- The 2026 World Cup group draw used is based on the official draw held in December 2024.
- Football is inherently unpredictable — treat all outputs as statistical estimates, not guarantees.
